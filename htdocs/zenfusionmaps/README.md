ZenFusion Maps
==============

A Google Maps module for Dolibarr.

Functions
---------

Simply adds a nice icon and a link to Google Maps to addresses for:
- Thirdparties
- Contacts
- Members

Prerequisites
-------------

- PHP ≥ 5.3.0
- Dolibarr ≥ 3.5.0
